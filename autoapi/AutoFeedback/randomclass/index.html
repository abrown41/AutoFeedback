randomvar

generic class for random variables.
class AutoFeedback.randomclass.randomvar(expectation, dist='normal', variance=0, vmin='unset', vmax='unset', isinteger=False, meanconv=False, dof=-1, limit=-1, transform=None, nsamples=1)[source]¶

generic class for random variables. allows you to test if students are sampling from a distribution.

Attributes

the expectation/s for your random variable/s

the lowest value the random variable can take

isintegerbool/list of bools

True if the random variable must take an integer value

This is what is printed if the print method is executed on an instance of randomclass: it’s defined as it is so that when AutoFeedback prints out the expected value of a given variable we get something other than ‘<AutoFeedback.randomclass.randomvar>’
